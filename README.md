## url
- cli: `http://locahost:8080/supermarket`

## database
- cmd: `mysql -uroot -p`
- cmd: `root`
- cmd: `create database supermarket character set utf8mb4;`
- cmd: `use supermarket;`
- cmd: `grant all on supermarket.* to 'supermarket'@'localhost' identified by 'supermarket';`
- cmd: `flush privileges;`
